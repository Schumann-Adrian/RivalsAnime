# 🎌 RivalsAnime 🎌
> **RivalsAnime** es una aplicación de escritorio desarrollada en C# que combina la gestión de personajes con un sistema de combate estratégico 2v2 por turnos.

---

## 🚀 Flujo de la Aplicación

El sistema sigue una lógica intuitiva diseñada para diferentes perfiles de usuario:

### 1️⃣ Pantalla de Inicio
Al ejecutar el proyecto, serás recibido por la pantalla principal con el título **RivalsAnime**. 
* Haz clic en el botón **"Comenzar"** para entrar al sistema.

### 2️⃣ Acceso y Seguridad (Auth)
* **Registro:** Si es tu primera vez, dirígete a la pestaña de registro. Es fundamental crear una cuenta para persistir tus datos.
* **Login:** Una vez registrado, inicia sesión con tus credenciales para acceder a las funciones personalizadas.

### 3️⃣ Panel Principal (Dashboard)
Dependiendo de tu rol, verás opciones distintas:
* **🛡️ Perfil Administrador:** Acceso completo al **CRUD**. Podrás Crear, Leer, Actualizar y Eliminar personajes de la base de datos/historial.
* **⚔️ Perfil Usuario:** Acceso directo a la zona de **Combate**.

### 4️⃣ Visualización de Personajes
Antes de luchar, puedes explorar la galería:
* Consulta las **Stats** detalladas (Ataque, Defensa, Velocidad, etc.).
* Visualiza el **Arte/Imagen** oficial de cada personaje.

### 5️⃣ Arena de Combate
¡Llegó el momento de la verdad!
* Sistema de combate **2v2**.
* Lucha por **turnos** estratégicos.
* Elige tus mejores personajes y domina al rival.
